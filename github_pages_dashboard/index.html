Data Schema & Upload

This dashboard reads /data/stats.json. See schema.json for the full data model and examples. You can POST new CSV/JSON to an ingestion script on your server to update stats.json automatically.

This is a static starter template. To activate AI features, deploy a small serverless function (example included in /ai_placeholder) that consumes stats.json and returns insights. See README.
